# OpenCode Context Navigation

This directory provides the local context loaded by agents configured in `opencode.json`.

Start here:

- `core/navigation.md` - shared standards and workflows used across tasks
- `project/navigation.md` - repo-specific conventions for `architecture/infrastructure`

Optional compatibility areas:

- `openagents-repo/quick-start.md` - compatibility stub, not specific to this repo
- `system-builder-templates/navigation.md` - template-system references for builder agents
- `ui/web/react-patterns.md` - placeholder UI guidance when frontend work exists
